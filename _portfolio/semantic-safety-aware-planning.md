Built a hierarchical planning system combining a vision-language model (VLM) for high-level decision-making in an options-over-MDP formulation with a DWA-based low-level controller. Used risk-inducing objects to create grounded visual context and benchmark VLM behavior in uncertain environments.

Undergraduate thesis: [view thesis](https://drive.google.com/file/d/1YDo8a117sccprqWSJaHE4-YgctgOKO83/view?usp=sharing).

Work in preparation for publication.
